CSS Selectors & Specifity

CSS Selector Example Explanation
Select by #id #firstname #id is the most specifc selector as only one element will have that specific #id. Only the element with the specific #id of id="firstname" will be affected by this selector. It will overwrite any existing CSS from other selectors such as .class or element as they are lower in the selector heirarchy.
Select by .class .intro .class ranks second in the select heirarchy, it is more specific than an element but can be overwritten by the #id. .class is a great way to style multiple elements at a time, as the selector can be added to multiple items, allowing you to reuse the same styling any elements you add the class="intro" too.
Select by Element p element is the lowest selector in the heirarchy, using it will affect any element with that tag but the CSS from an element selector can easly be overwritten using a .class or #id selector as they are more specific. It is easy way to add unversal styles to tags, any

element will be effected by this selector

All selectors can be used together to create more specific slectors, to target single areas or specific elements of a page p .class will target every p element with a specific class. #id .class will target only the elements that have both the specifi #id pluss the extra class.
